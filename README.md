# Яндекс.Директ — Дэшборд

Интерактивный дашборд для визуализации Excel-выгрузок статистики Яндекс.Директа. Single-file SPA на Chart.js + SheetJS, без бэкенда и сборщика.

## Ссылки

| Площадка | URL |
|---|---|
| **GitHub Pages** | https://perejaslav.github.io/ydd/ |
| **Netlify** | https://yandex-direct-dashboard.netlify.app |

## Как использовать

1. Откройте дашборд по ссылке выше (или `dashboard.html` локально).
2. Нажмите **«Загрузить Excel»** и выберите файл выгрузки Яндекс.Директа.
3. Дашборд автоматически определит метрики и построит KPI, графики и таблицу.

### Сравнение периодов

1. Включите тумблер **«Сравнение периодов»** в header.
2. Загрузите два файла: **A** (основной) и **B** (для сравнения).
3. Дашборд покажет дельту по всем метрикам: KPI с цветовой маркировкой, 3 графика (grouped bar, diverging bar, scatter), таблица с Δ%.

## Возможности

- Авто-определение колонок (70+ известных метрик Яндекс.Директа, fallback для неизвестных)
- KPI-карточки (показы, клики, расход, CTR, конверсии, CPC/CPA…)
- 8 графиков: bar (расход + клики), CTR, CPC, doughnut (по измерениям), отказы, конверсии, топ-15 регионов, bubble (CTR vs CPC)
- Детальная таблица с сортировкой по клику
- Фильтры по измерениям (кампания, группа, устройство, регион…)
- Toast-нотификации об ошибках
- Тёмная тема, адаптивный дизайн

### Сравнение периодов (два файла)

- KPI с дельтой (▲ рост / ▼ падение, цвет по направлению метрики)
- Grouped bar: A vs B по топ-15 кампаний
- Diverging bar: дельта % с цветовой маркировкой
- Scatter: A по X, B по Y (диагональ = без изменений)
- Таблица: кампания + A/B/Δ% для до 3 ключевых метрик, блоки «Только в A / Только в B»
- Фильтр по кампании, применяется к обоим датасетам

## Технологии

- [Chart.js 4.4.7](https://www.chartjs.org/) — графики
- [SheetJS 0.18.5](https://sheetjs.com/) — парсинг XLSX
- Vanilla JS, CSS Grid, без сборщиков

## Структура

```
dashboard.html    — весь код (разметка + стили + скрипты, ~1400 строк)
index.html        — редирект на dashboard.html (GitHub Pages)
netlify.toml      — конфиг Netlify (редирект / → /dashboard.html)
context.md        — контекст проекта (архитектура, баги, точки расширения)
PLAN.md           — исходный план починки формата «Мастер отчётов»
docs/             — спека и план реализации сравнения периодов
```

## Локальный запуск

```bash
# Через любой HTTP-сервер (файл использует CDN, нужен HTTP)
npx serve .
# или
python -m http.server 8000
```

Откройте `http://localhost:8000` (или `http://localhost:8000/dashboard.html`).
